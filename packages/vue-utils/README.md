# Vue Utils

A small package that contains common utilities around Vue, Vue Router and vue-i18n used in all my projects.

## 📃 Prerequisites
- **[Bun](https://bun.sh)**: >= 1.3.10

## 📦 Building
- Use the command `bun install` in the root directory to install the required dependencies.
- Run `bun --cwd packages/vue-utils build` to build the package.
