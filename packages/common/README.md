# Common

A small package that contains common application helpers used across my projects, including Vue, Vue Router and vue-i18n integrations.

## 📃 Prerequisites
- **[Bun](https://bun.sh)**: >= 1.3.10

## 📦 Building
- Use the command `bun install` in the root directory to install the required dependencies.
- Run `bun --cwd packages/common build` to build the package.
